# 3D ASCII Black Hole

A real-time 3D ASCII-art black hole simulation rendered in the browser.

[Live Demo](https://vanie2300.github.io/web)

## Features

- Accretion disk with gravitational lensing and shadows
- Photon ring and event horizon
- Animated background stars
- Smooth rotation (auto or drag with mouse/touch)
- FPS counter

## How to use

Open `index.html` in a browser, or visit the [live demo](https://vanie2300.github.io/web).

- **Drag** to rotate the view
- **Click/tap** to show controls hint
- Works on desktop and mobile
